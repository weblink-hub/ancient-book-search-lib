## 国风短句与诗词摘抄素材库
很多人喜欢收集各类优美国风短句，用于文案创作、读书笔记、摘抄积累、社交文案使用。市面上零散的短句素材杂乱无章，缺少系统分类，为此整理海量古风句子、诗词节选、国风温柔文案，全部按场景、意象、题材划分，方便按需检索摘抄。

### 一、素材内容分类
1. 四季风物短句：春樱夏雨、秋霜冬雪相关唯美诗句片段；
2. 山河风光文案：大漠、江南、江海、名山写景摘抄；
3. 人间烟火短句：市井、三餐、田园、闲居温柔文字；
4. 离愁抒怀诗句：送别、思念、思乡、感怀类节选；
5. 草木花木意象：梅兰竹菊、桃花垂柳、荷桂专属短句；
6. 少年温柔文案：意气、成长、清风明月治愈句子；
7. 怀古风雅文字：古迹、旧岁、岁月沧桑国风短句；
8. 修身静心短句：独处、淡然、释怀治愈古风文案。

### 二、素材库优势
1. 全部摘抄自古典诗词、明清小品、名家散文，无网络低俗文案；
2. 短句篇幅短小，适合直接复制使用，适配笔记、短视频、朋友圈；
3. 标注原文出处，方便需要完整诗文的读者溯源查阅；
4. 分类清晰，不用通篇翻找，按需求直接定位对应题材素材。

### 三、适用人群
1. 文案创作者，短视频、图文内容古风素材储备；
2. 学生群体，语文摘抄本、作文优美句子积累；
3. 国风爱好者，日常收藏温柔治愈古风文字；
4. 写作新手，借鉴古典文字意象提升文笔质感。
